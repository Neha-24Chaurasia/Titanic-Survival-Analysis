# 🚢 Titanic-Survival-Analysis 
It is a ML project to predict Titanic passenger survival using feature engineering and classification models
## ⚙️ Models and Accuracy
1. Logistic Regression- ### 81%
2. Random Forest- ### 82.68%
## 🧠 Feature Engineering
- Missing Values Handling
- Outliers Detection
- Encoding Categorical Variables
- Created features like Family Size, IsAlone
## 📈 Final Output
- Passenger Survival List generated
- Survived Passenger list in different file titanic_survived_passengers.csv using Logistic Regression
## 🚀 Tech Stack 
Python, Pandas, Numpy, Scikit-learn
